🔍 Reconnaissance

The process of gathering information about a target before attacking.

Key Concepts

  • Passive Recon (OSINT, Google dorking)
  • Active Recon (ping, traceroute)

Notes

Passive recon avoids detection, while active recon interacts directly with the target system.

📡 Scanning

Identifying live systems, ports, and services.

Key Concepts

  • Port scanning
  • Service/version detection
  • OS fingerprinting

Notes

Tools like Nmap help discover vulnerabilities by mapping open ports.

🧠 Enumeration

Extracting detailed information from systems.

Key Concepts

  • User accounts
  • Network shares
  • Services and roles

Notes

Enumeration is more intrusive and often follows scanning.

💣 Exploitation

Gaining access by exploiting vulnerabilities.

Key Concepts

  • Buffer overflows
  • Web exploits (XSS, SQLi)

Notes

This phase uses known vulnerabilities or custom exploits.

🕵️ Post-Exploitation

Maintaining access and gathering further data.

Key Concepts

  • Privilege escalation
  • Persistence
  • Data exfiltration

Notes

This stage focuses on control and long-term access.

🛠 Tools

  • Nmap
  • Wireshark
  • Metasploit
  • Burp Suite
